# Breanna Morkrid — Technical Portfolio

A clean, responsive, and lightweight developer portfolio designed to showcase my software engineering projects, technical skills, and professional journey. Built from scratch using semantic HTML5, modern CSS, and vanilla JavaScript.

https://breanna1202.github.io/Portfolio/

---

## 🚀 Features

*   **Responsive Design:** Fully optimized for all screen sizes, from mobile devices to ultrawide monitors, using CSS Grid and Flexbox.
*   **Privacy-Conscious Architecture:** Designed with zero tracking cookies or invasive third-party scripts.

---

## 🛠️ Tech Stack & Concepts Demonstrated

*   **Structure:** Semantic HTML5 for structural clarity, accessibility (ARIA attributes), and clean SEO mapping.
*   **Styling:** Modern CSS (Custom Properties/Variables, Flexbox, Grid, and fluid typography).
*   **Interactivity:** Vanilla JavaScript (ES6+) focusing on clean DOM manipulation.
